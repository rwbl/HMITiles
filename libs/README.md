# Libraries

Libraries when working with **HMITiles**.

---

- HMITiles.b4xlib
	- Main HMI Tiles library.
	- Copy to the B4A or B4J additional libraries folder.
		- Ensure to backup your B4A/B4J additional libraries folder first.

- HMITilesX.b4xlib
	- HMI Tiles Extensions library.
	- Copy to the B4A or B4J additional libraries folder.
		- Ensure to backup your B4A/B4J additional libraries folder first.
	- Depends on the library HMITiles.
	- **Under development**
