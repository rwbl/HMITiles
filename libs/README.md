# Libraries

Libraries when working with **HMITiles**.

---

- HMITiles.b4xlib
	- Main HMI Tiles library.
	- Copy to the B4A or B4J additional libraries folder.
- B4R-Libraries.zip
	- rBLE Server.
	- Unpack the archive and copy to the B4R additional libraries folder.
- B4A-Libraries.zip
	- BleGattManager.b4xlib - B4A BLE manager used by B4A examples controllig BLE devices.
	- Unpack the archive and copy to the B4A B4J additional libraries folder.
