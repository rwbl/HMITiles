# GetStarted
Welcome to the HMITiles entry point. 
This folder provides isolated, easy-to-follow examples for every component in the library.
What's Inside

    Component-Specific Demos: Each subfolder contains a standalone example for a single HMITile.
    Reference Code: Use these snippets to understand the required properties, data structures, and configuration for each tile.

Simply browse the subfolders to see a specific tile in action and copy the implementation logic into your own project.

---

## Platform Notes

Most examples are available for **B4J**.  
Selected examples are also provided for **B4A**, but coverage is not yet complete.

Additional B4A examples may be added over time.
