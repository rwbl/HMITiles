# Additional Libraries required for the Examples

Additional libraries when working with the **HMITiles** examples.

Ensure to backup your additional libraries folder prior copying the files below.

---

- B4R
	- rNimBLEServer - B4R BLE GATT manager for the ESP32 MCU.
	- rConvert.b4xlib - Conversion helpers.
	- Copy the folder to the B4R additional libraries folder.
